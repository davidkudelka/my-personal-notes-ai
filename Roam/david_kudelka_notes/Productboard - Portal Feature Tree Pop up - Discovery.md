    * ## Some basic findings
      * ### Portal add feature dropdown
        * **Name of the component** - PortalAddFeatureDropdown
        * **What kind of data goes in** - Feature id's with view access for current user
        * **Where is it used** - 3 places
          * Portal Add Feature button, when portal view is empty and there is button to add feature in the middle
          * Floating button - left bottom - you can click on it and add feature to portal
          * When you see sections on portal - on the line you can click plus button, which will give you pop up to add feature to some section
