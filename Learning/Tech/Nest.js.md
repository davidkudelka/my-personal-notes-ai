  * Backend framework to help you build scalable server-side applications
  * Description
    * Controllers - handling requests and responses
    * Providers - providing some functionality
    * Modules - putting together different controllers and providers, for example you can have orders module that bundles together all order providers and controllers
    * Middlewares - typical middleware
  * Plus
    * Support for GraphQL
    * You can use prisma 2 together with Nest.jS
    * Gives you set rules about structure of your project - more easily to follow onion architecture style - to properly follow business logic separation
    * CLI - easy tool get you started
  * Cons
    * Might be missing support for some important features - might be solvable through some custom controllers or providers
    * You are forced to use concepts set by Nest.js
  * Summary
    * Might be worth trying out, this framework got recently a lot of popularity, because it simplifies a lot of issues around backend development - might very useful for traditional backend applications
