    * ## Basics
      * The Daily notes is good way where to start putting some information into the Roam
      * We can then create some pages for those information obtained in the daily notes
      * ### The 4 Ways to Link Between Pages
        * First one is normal double bracket link using [[]]
        * You can use as well tag links #[[This is testing link]]
          * Functionality of this tag is absolutely same as double brackets, the only difference is visual
        * You can create attributes with (double :) mark and this acts exactly same as previous examples, but it is adding another visual difference between different types of references
        * You can use alias by {{}} and put alias in there
          * {{alias: [[2020-12-26]] Some day}}
      * ### Increasing you Roam productivity with the Sidebar

      * ### Filtering and Navigating Page Relations
        * You can filter the page based on different things in the top right corner with filter icon
      * ### Todos and Task Management
